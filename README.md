This bot automaticaly fixes [Twitter](https://twitter.com) (or [X.com](https://x.com)), [Instagram](https://instagram.com), [TikTok](https://tiktok.com) or [Reddit](https://reddit.com) links

### Commands:
1. /toggle
2. /status
3. /shutdown
